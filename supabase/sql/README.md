# Trimax Supabase SQL

Run these scripts in the Supabase SQL editor when the app needs the matching database structure.

## Workspace Access

1. Run `2026-05-24-business-users.sql`.
2. Create Auth users for Robbie and Lyuba in Supabase Authentication.
3. Open `2026-05-24-seed-business-users.sql`.
4. Keep Robbie's email as `robbie@rnlcreations.com` if that is the login email.
5. Replace `lyuba@justkleen.com` with Lyuba's real login email.
6. Run `2026-05-24-seed-business-users.sql`.

After this, Robbie lands in the R&L workspace and Lyuba lands in the JUST KLEEN workspace.
